1. Quickstart

TokenMaster is a drop-in OpenAI-compatible API gateway. You get one master key that works across all 17+ supported models — GPT-4, Claude, Gemini, Zhipu GLM-4-Plus, DeepSeek, Qwen, Mistral, Llama 3, and more. No separate accounts needed.

Tip: If you've used the OpenAI SDK before, you already know 90% of TokenMaster. Just swap the base URL and API key.

2. Installation

Install the OpenAI SDK for your language of choice. TokenMaster speaks the exact same protocol.

Python

pip install openai

Node.js

npm install openai

cURL

No install needed. TokenMaster works with any HTTP client.

3. Authentication

All requests authenticate with a Bearer token. After purchase, you'll receive your master key via email.

Authorization: Bearer TM-xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
Keep it secret: Never expose your key in client-side code or public repos. Use a backend proxy for production apps.

4. Your first request

Point your OpenAI SDK to TokenMaster's base URL and send a chat completion:

Python

from openai import OpenAI

client = OpenAI(
    api_key="TM-xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    base_url="https://api-tokenmaster.com/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello, world!"}]
)
print(response.choices[0].message.content)

Node.js

import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'TM-xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx',
  baseURL: 'https://api-tokenmaster.com/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello, world!' }]
});
console.log(response.choices[0].message.content);

cURL

curl https://api-tokenmaster.com/v1/chat/completions \
  -H "Authorization: Bearer TM-xxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello, world!"}]
  }'
