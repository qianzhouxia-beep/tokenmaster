API Reference

Chat Completions

POST/v1/chat/completions

Create a chat completion. The request body matches OpenAI's chat.completions.create schema exactly.

Request body

FieldTypeRequiredDescription
modelstringyesModel id. See /models for the full list.
messagesarrayyesArray of {role, content}. role is "system" / "user" / "assistant".
temperaturenumberno0.0–2.0. Default 1.0. Higher = more random.
max_tokensintegernoCap on tokens to generate. Model-dependent.
streambooleannoIf true, return SSE stream of completion chunks.
top_pnumbernoNucleus sampling. 0.0–1.0. Default 1.0.
stopstring | arraynoStop sequences. Up to 4.

Example request

POST https://api-tokenmaster.com/v1/chat/completions
Authorization: Bearer TM-xxxxxxxx
Content-Type: application/json

{
  "model": "gpt-4o",
  "messages": [
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "Explain quantum entanglement in 2 sentences."}
  ],
  "temperature": 0.7,
  "max_tokens": 200
}

Example response

{
  "id": "chatcmpl-abc123",
  "object": "chat.completion",
  "created": 1717929600,
  "model": "gpt-4o",
  "choices": [
    {
      "index": 0,
      "message": {
        "role": "assistant",
        "content": "Quantum entanglement is a phenomenon where two particles become correlated such that the state of one instantly influences the state of the other, regardless of distance. It's a key resource in quantum computing and quantum cryptography."
      },
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "prompt_tokens": 24,
    "completion_tokens": 41,
    "total_tokens": 65
  }
}

Streaming

Set stream: true to receive Server-Sent Events. Each event is a JSON chunk followed by \\n\\n.

event: data
data: {"id":"chatcmpl-abc123","object":"chat.completion.chunk","choices":[{"delta":{"content":"Quantum"},"index":0}]}

event: data
data: {"id":"chatcmpl-abc123","object":"chat.completion.chunk","choices":[{"delta":{"content":" entang"},"index":0}]}

event: data
data: [DONE]

Embeddings

POST/v1/embeddings

Generate vector embeddings for semantic search, RAG, clustering.

FieldTypeRequiredDescription
modelstringyesCurrently text-embedding-3-small, text-embedding-3-large, text-embedding-ada-002.
inputstring | arrayyesText or array of texts to embed.
encoding_formatstringnofloat (default) or base64.

Errors

Errors return a JSON body with a code and message field. See /docs#errors for the full table.

{
  "error": {
    "code": "insufficient_tokens",
    "message": "Your account is out of tokens. Please purchase more at https://api-tokenmaster.com/landing#pricing.",
    "type": "billing_error"
  }
}
